前言

现在使用简化风格的博客越来越受欢迎，由于本人之前没有创建过个人微博，2019 开年之际，着手研究个人博客的搭建，经过一周的玩命折腾，终于将我的博客搭建完成了，那后面的事情就是写博客了，看到别人的博客风格、排版各种高大上，羡慕不已，一心模仿，经过网上的搜索，才知道是用 Markdown 写的，进过我几天的学习，特将 Markdown 的基本语法记录在案。一方面是加深自己的记忆，另一方面也是为其他观看我博客的同学做少许贡献，希望对大家有所帮助！

### 正文篇
正文只需正常编写即可，如需首行空两个字，直接敲空格就可以了，一个空格是一个字节，一个中文汉字是四个字节；如需换行，在需要换行的地方敲两下空格，然后回车
