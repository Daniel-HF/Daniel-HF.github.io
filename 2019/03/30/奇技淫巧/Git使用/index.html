git 简介

首先我先来说说git是什么，说到这个问题那不得不说说git的来历。
早在我还没有出生的时候，一个名叫Linus 的牛人就创建了开源的Linux，这个由全世界志愿者
程序员开发的至今已经成为了各大公司的服务解决方案，知道有多牛了吧！
随着越来越多的人加入，代码的版本控制成了一个很严重的问题，一开始都是由各个参与的志愿者将源文件发送给Linus,由他通宵手工合并代码。后来发现他自己快累死了，必须得找个办法解决一下，各种选择后最终决定使用BitKeeper,这是一个商用的版本控制系统，这家公司也还挺有心，让他们免费使用的，不用给钱。这似乎是个好事，但好景不长，在2005年，一个
